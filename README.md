# Docker Todo Application

A complete Cloud & DevOps portfolio project demonstrating containerization, CI/CD automation, and cloud deployment using Flask, MySQL, Docker, GitHub Actions, and Microsoft Azure.

---

## Project Overview

This project is a Dockerized Todo Application built with Flask and MySQL. The application is containerized using Docker Compose, automated with GitHub Actions CI/CD, and deployed on a Microsoft Azure Virtual Machine using Nginx Reverse Proxy.
